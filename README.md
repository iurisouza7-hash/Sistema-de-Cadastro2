# 🧾 Sistema de Cadastro (CRUD em Python)

Projeto de um sistema de cadastro de usuários desenvolvido em Python, utilizando JSON como forma de armazenamento de dados.

---

## 🚀 Funcionalidades

* ➕ Adicionar usuário
* 📋 Listar usuários 
* ✏️ Editar usuário *(em desenvolvimento)*
* 🗑️ Deletar usuário *(em desenvolvimento)*

---

## 🛠️ Tecnologias utilizadas

* 🐍 Python
* 📁 JSON

---

## 📦 Como funciona

O sistema roda diretamente no terminal e permite cadastrar usuários com as seguintes informações:

* Nome
* Idade
* Email
* Telefone
* Cidade
* Senha

Todos os dados são armazenados no arquivo `usuarios.json`, funcionando como um banco de dados simples.

---

## ▶️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/sistema-de-cadastro.git
```

2. Acesse a pasta do projeto:

```bash
cd Sistema-de-Cadastro
```

3. Execute o programa:

```bash
python main.py
```
